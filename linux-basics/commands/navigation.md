# Linux Navigation Commands

This file covers basic Linux navigation commands used to move around the file system and understand your current location in a Linux environment.

These are essential commands for working in any Linux server or terminal.

---

## 1. pwd (Print Working Directory)

Shows the full path of the directory you are currently in.

```bash
pwd
```

## 2. ls (List Files)

Shows all the folders and files of the directory you are currently in.

```bash
ls
```

## 3. cd (Change Directory)

Change the directory you are currently in.

```bash
cd
```

## 4. tree

Recurivsely list the contents of a folder.

```bash
tree
```
